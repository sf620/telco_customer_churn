# Telco Customer Churn

<!-- [Telco customer churn (11.1.3+)](https://community.ibm.com/community/user/blogs/steven-macko/2019/07/11/telco-customer-churn-1113) -->

Machine learning project for predicting customer churn using tabular data and supervised classification.

**Python version required:** 3.12

## Repository Contents

`telco_customer_churn.ipynb` — data analysis, preprocessing, training, and model evaluation notebook

`report.md` — project report

`requirements.txt` — minimal Python dependencies derived from notebook imports

## Quick Start

```bash
# Clone repository
git clone https://github.com/sf620/telco_customer_churn
cd telco_customer_churn

# Create and activate a virtual environment
python -m venv venv
source venv/bin/activate  # macOS/Linux

# Install dependencies from requirements.txt
pip install -r requirements.txt
pip install jupyter

# Launch Jupyter
jupyter notebook
```

Open `telco_customer_churn.ipynb` and run all cells.
